## Managing Tickets

### Dashboard Overview

**Statistics Cards** (top of page):
- Open Tickets: New tickets awaiting action
- In Progress: Tickets currently being worked on
- Completed: Finished tickets awaiting review
- Total Tickets: All tickets in the system

### Viewing a Ticket

1. Click on any ticket card to open details
2. You'll see:
   - Full description
   - All updates and comments
   - Status and assignment
   - Complete history

### Updating a Ticket

**Change Status:**
- Open the ticket
- Click the Status dropdown (right side)
- Select new status:
  - Open: Ticket created, not started
  - In Progress: Currently working on it
  - Completed: Work finished, awaiting review
  - Closed: Fully resolved

**Reassign a Ticket:**
- Open the ticket
- Click "Assign To" dropdown
- Select new team member
- They'll receive an email notification

**Add an Update:**
- Open the ticket
- Scroll to "Updates & Comments" section
- Type your update in the text box
- Click "Add Update"
- Everyone can see the update history

**Send Email Update:**
- Open the ticket
- Click "üìß Send Email Update" button
- Email sent to assigned person with current status

3. **Priority Filter**: Show only urgent/high/medium/low tickets
4. **Search Box**: Type anything (title, description, ticket ID)

## Common Workflows

### Emergency Repair (Urgent)

1. Create ticket with "Urgent" priority
2. Assign to appropriate team member
3. Set due date to today
4. Team member receives immediate email
5. Track progress through updates
6. Close when resolved

### Routine Maintenance

1. Create ticket with "Low" or "Medium" priority
2. Assign to Kevin or team
3. Set reasonable due date
4. Team member works on it
5. Updates added as work progresses
6. Mark completed when done

### Multi-Step Projects

1. Create main ticket
2. Add detailed breakdown in description
3. Use updates section to track each step
4. Update status as you progress
5. Add photos or details in updates
6. Close when all steps complete

## Troubleshooting

### "Can't log in"
- Try incognito/private browser mode
- Clear browser cookies
- Verify you're using your Outlook email

### "Tickets not saving"
- Refresh the page
- Check internet connection
- Try different browser

### "Email not received"
- Check spam folder
- Verify email permissions were granted
- Ask Kevin to check system logs

### "Can't find a ticket"
- Clear all filters
- Use search with ticket ID
- Check if it was deleted
